SampleSocketClient
====

## Overview

Socket client sample code for C#.

## Requirement

.NET Framework 4.8

## Usage

<img width="387" alt="disp" src="https://user-images.githubusercontent.com/34742256/150057017-07f92cd9-e39e-41e8-8d66-eb34e96797cb.png">

### ① Server Address Box

Enter server IPv4 Address.

### ② Server Port

Enter TCP port to connect.

### ③ Connect Button

Connect to specified server port.

### ④ Disconnect Button

Disconnect from server.

### ⑤ Send Data

Enter TCP payload.

### ⑥ Send Button

Send TCP data to server.

### ⑦ Message Box

Show the TCP payload from server, error message, etc...
